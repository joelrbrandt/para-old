Para




Thanks for trying out Para.

Here's the basics

  • Use the pen tool to draw shapes using bezier curves.
  • Use the select tool to select and manipulate shapes.
  • Pan and zoom by holding down the command key (âŒ˜) and clicking and dragging for pan, or using the mouse wheel to zoom.
  • Save a version of your drawing by entering a name in the righthand panel and selecting save. You can download a version of your saved file to your computer by selecting save to computer.
  • You can load a locally saved file by selecting it from the dropdown menu. To load in a file saved on your computer, select the upload file from desktop option.
  • Export an svg of your file with the download svg option, and edit it in your favorite vector software. A heads up, you will not be able to re-import this svg into Para after you're done.

Here's the important stuff

  • Create distributions of shapes by selecting a shape while holding the alt key to pull up the modifier menu and select add behavior.
  • Selecting copy will create a single copy of your shape.
  • Selecting linear distribution will create a linear pattern of shapes.
  • Selecting radial distribution will create a radial pattern of shapes.
  • You can adjust distributions by dragging the red and green shapes.
  • Create a distribution of shapes along a path by dragging a shape over a path while holding SHIFT and releasing it.
  • To edit a distribution, double click on any shape in it. This will move you down a level in the heirarchy of the drawing (sort of like moving between groups in illustrator). Double click outside the distribution to move back up.
  • Create multi-level distributions and use the double clicking mechanism to edit them
  • Changing one shape or distribution will change all the copies of it accordingly. This is a feature, not a bug.

Here's the bad news

  • Para is a prototype, meaning it's very rough and there's a lot of bugs. We're sorry in advance. Here's someone you can complain to.
  • Para is not optimized, so if you create a billion shapes in your drawing, it's going to start moving very slowly

Here's the good news

  • Para is open-source software. Take a look: github.com/pixelmaid/para
  • We're continuing to work to add features, fix bugs, and make Para awesome. If you have ideas for features you'd like to see, we'd love to hear from you.
